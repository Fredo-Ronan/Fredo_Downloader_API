Welcome to Fredo Downloader API

What is this? üßê

This is an API for download any instagram or youtube videos using just the share link. For now this API only support instagram video download, any other post you can try it yourself if it's works, and for the youtube video download it's still in development progress.

How to use it? ü§î

To use this API is simply using the same link follow by some parameters to download things.
The link structure is like this : 

https://fredo-downloader-api.vercel.app/api/instagram-download?url={your_instagram_share_link}

Example üßë‚Äçüíª

The instagram share link or the link to instagram video/reel : 

https://www.instagram.com/reel/C2IDZSNs6v8/

Concat with the link using the query parameter like this: 
https://fredo-downloader-api.vercel.app/api/instagram-download?url=https://www.instagram.com/reel/C2IDZSNs6v8/
